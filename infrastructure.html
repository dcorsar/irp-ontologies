The IRP Infrastructure ontology

The IRP Infrastructure ontology (Ontology) VC Definition In use

The IRP Infrastructure Ontology combines various existing concepts to describe physhical infrastructure related to public transport (e.g. roads, railways) developed during the Informed Rural Passenger project at the dot.rural Digital Economy Hub, University of Aberdeen, UK supported by the award made by the RCUK Digital Economy programme; award reference: EP/G066051/1. The ontology is reuses concepts from the UK Department for Transport Traffic vocabularly and OpenGeoData.

Geo point (OWL Class) Definition In use

A convience class for representing geo-location information about a point, possibly using multiple co-ordinate systems such as the UK Ordnance Survey Easting and Northing, or the WGS84 Longitude/Latitude encoding.

Infrastructure point (OWL Class) Definition In use

A point (or node) on a piece of infrastructure related to public transport; for example, a road can be defined as an ordered list of such points (each with geo-coordinates), with each pair representing a link (or section) of that road.

Definition

The URI of this class is http://www.w3id.org/abdn/irp/infrastructure/InfrastructurePoint

This class is a super class of Road point
